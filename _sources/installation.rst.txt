Installation
============

From PyPI:

.. code-block:: bash

   pip install mimiq-qiskit

From source:

.. code-block:: bash

   git clone https://github.com/qperfect-io/mimiq-qiskit.git
   cd mimiq-qiskit
   uv sync

Runtime dependencies:

- Python >= 3.10
- `mimiqcircuits <https://pypi.org/project/mimiqcircuits/>`_ >= 0.24
- `qiskit <https://www.ibm.com/quantum/qiskit>`_ >= 2.0
- numpy >= 1.24
