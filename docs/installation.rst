Installation
============

From PyPI (once published):

.. code-block:: bash

   pip install mimiq-qiskit

From source:

.. code-block:: bash

   git clone https://gitlab.qperfect.io/development/mimiq/mimiq-qiskit.git
   cd mimiq-qiskit/mimiq-qiskit-python
   uv sync

Runtime dependencies:

- Python ≥ 3.10
- `mimiqcircuits <https://gitlab.qperfect.io/development/mimiq/mimiqcircuits-python>`_ ≥ 0.21
- `qiskit <https://www.ibm.com/quantum/qiskit>`_ ≥ 2.0
- numpy ≥ 1.24
